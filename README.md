Ntplib
------------

An ntp packet parsing library written in Rust.


Usage
-----

Add this to your `Cargo.toml`:

```ini
[dependencies]
ntplib = { git = "https://github.com/JeffBelgum/ntplib" }
```

and this to your crate root:

```rust
extern crate ntplib;
```
